## [1.0.0] - 2026-05-28

### Added

- Initial release of the prompt-coach skill.
- SKILL.md with trigger logic, behavior spec, output format, and tone guidelines.
- Full reference doc covering all 12 prompting best practices from Anthropic's official guide.
- Detection patterns with priority-ordered matching for each practice.
- 12 eval test cases — one per best practice — for verifying skill behavior.
- GitHub-ready README with installation instructions for Claude Code, Claude.ai, and Cowork.
